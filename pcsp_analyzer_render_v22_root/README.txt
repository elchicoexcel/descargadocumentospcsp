PCSP Analyzer v2.0 - backend real

1) Instalar dependencias:
   pip install -r requirements.txt

2) Ejecutar:
   python app.py

3) Abrir en el navegador:
   http://127.0.0.1:5000

IMPORTANTE:
Esta versión debe ejecutarse con backend local. Un HTML abierto directamente en el navegador no puede descargar documentos de contrataciondelestado.es de forma fiable por CORS y redirecciones.
